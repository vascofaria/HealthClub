# HealthClub
Software Testing and Validation 2020/21 Project

## To compile and run:

```
mvn compile exec:java
```

## To compile and run with debug messages:

```
mvn compile exec:java -Ddebug
```

## To Compile
```
mvn compile
```

## To Test
```
mvn test
```

## To Compile and Test
```
mvn compile test
```

## To Clean Project

mvn clean
